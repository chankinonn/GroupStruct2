# GroupStruct2

**GroupStruct2** is an R package that launches a Shiny application for analyzing morphological, meristic, and mixed trait data to explore species structure, perform allometric corrections, visualize trait distributions, and conduct multivariate analyses like PCA and MFA.


---

## 🔧 Installation

```r
# Install remotes if needed
install.packages("remotes")

# Install GroupStruct2 from GitHub
